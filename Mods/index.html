Requerimientos¶

  • Descargar:

    • Fabric

      • Instalador

    • Pack de Mods

      • Descargar (Google Drive)

Proceso de Instalación¶

  1. Descargar y ejecutar el Instalador de Fabric, en la ventana colocar las siguientes opciones y dar en Install.

  2. Ir a la carpeta del juego en C:\Users\(Tu Nombre de Usuario)\AppData\Roaming\.minecraft y verificar si se creo la carpeta mods, de lo contrario puedes crearla tu mismo.

  3. Descargar el Pack de Mods, luego, colocar todos los archivos .jar en la carpeta mods como en este ejemplo.

  4. Una vez instalado todo, iniciar el Launcher de Minecraft y buscar el Perfilcreado por Fabric, seleccionarlo y editar las preferencias de Nombre/RAM a gusto. Ejemplo, en el caso de SKLauncher.

    • El perfil creado podría verse diferente dependiendo del Launcher que se este utilizando, revisar con precaución.

  5. Iniciar Minecraft y verificar que todos los mods funcionen correctamente. La configuración de cada uno queda a criterio del usuario, pero siempre pueden utilizar el servidor de Discord para pedir Soporte de ser necesario.

Lista de Mods¶

    En la siguiente lista se especifican todos los mods que están autorizados para usarse en el servidor, y que vienen incluídos en el Pack.
  1. AppleSkin
    • Mejora el HUD de Minecraft agregando información adicional, como por ejemplo, la cantidad de saturación (comida) que regenera el objeto que tengas en la mano.
  2. Roughly Enough Items
    • Agrega una interfaz al Inventario con todos los objetos y recetas del juego.
  3. Xaero's Minimap
    • Agrega un minimapa personalizable al juego.
  4. Xaero's World Map
    • Agrega un mapa en pantalla completa personalizable al juego.
  5. Optifine
    • Optifine es un mod de optimización que trae gran cantidad de opciones para personalizar dentro del juego, muy recomendado para usuarios con PC de gama baja.
  6. OptiFabric
    • Permite la compatibilidad ente Optifine y Fabric.
  7. Architectury API
    • Es una API que permite la compatibilidad de ciertos mods. (No es necesario saber exactamente que hace).
  8. Fabric API
    • Es la API principal de Fabric, permite que se puedan utilizar mods.

Importante

  • Es altamente recomendable verificar que coloquen siempre la ruta correcta de donde tienen instalado Minecraft, por default siempre es C:\Users\(Tu Nombre de Usuario)\AppData\Roaming\.minecraft

  • Cualquier mod que no se describa en esta página no tiene autorización para utilizarse, y por lo tanto, es un motivo de ban dentro del Servidor de ser detectado.

  • Se recuerda a los usuarios, que el uso de cualquier tipo de Cheat, Hack, Exploit o Script dentro del servidor es un motivo de ban permanente sin posibilidad de apelación.
