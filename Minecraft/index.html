Requerimientos (Hardware)¶

  • Requerimientos Mínimos:

    • Procesador: Intel Core i3-3210 3.2 GHz / AMD A8-7600 APU 3.1 GHz o su equivalente.

    • Memoria RAM: 4GB DDR3/DDR4.

    • Tarjeta Gráfica: Intel HD Graphics 4000 / AMD Radeon R5 series compatible con OpenGL 4.4+

    • Disco: Al menos 1/10GB para guardar los Mundos, Texturas, Mods, entre otras cosas.

  • Sistema Operativo:

    • Windows: 7/8/10 64bits.

    • macOS: Cualquier OS X de 64bits que utilice Maverick 10.9 o una versión más reciente.

    • Linux: Cualquier distribución de 64bits de a partir de 2014.

  • Requerimientos Recomendados:

    • Procesador: Intel Core i5-4690 3.5GHz / AMD A10-7800 APU 3.5 GHz o su equivalente.

    • Memoria RAM: 8GB DDR4.

    • Tarjeta Gráfica: NVIDIA GeForce 700 Series | AMD Radeon Rx 200 Series con OpenGL 4.5+

    • Disco: Al menos 5/15GB para guardar los Mundos, Texturas, Mods, entre otras cosas.

  • Sistema Operativo:

    • Windows: 10 64bits.

    • macOS: macOS 10.12 Sierra.

    • Linux: Cualquier distribución de 64bits de a partir de 2014.

Requerimientos (Software)¶

  • Descargar e Instalar:

    • Java 8

      • Windows 64bits
      • macOS 64bits
      • Linux 64bits

    • Liberica Full JDK 8

      • Windows 64bits
      • macOS 64bits
      • Linux 64bits

    • Launcher Recomendado

      • SKLauncher 3.0

Proceso de Instalación¶

  1. Descargar e Instalar las dos dependencias principales Java 8 y Liberica Full JDK 8 dependiendo de tu Sistema Operativo y reiniciar el ordenador.

  2. Descargar e Instalar SKLauncher.

  3. Una vez abierto el Launcher, puedes cerrarlo y colocar el ejecutable que descargaste en C:\Users\(Tu Nombre de Usuario)\AppData\Roaming\.minecraft y crear un Acceso Directo en el Escritorio.

  4. De no tener una cuenta Premium de Minecraft, registrarse en la página de SKLauncher, indicar en el Launcher la opción de ¿No Premium? y loguearse utilizando el nombre de usuario que eligieron en la página.

    • En caso de tener una cuenta Premium, loguearse utilizando las credenciales normalmente.
  5. Una vez logueados dentro del Launcher, crear un Perfil nuevo, y escoger la versión actual del Servidor; en este caso, la 1.16.4.
  6. Se puede editar el Perfil para cambiar la cantidad de memoria RAM que puede utilizar el juego, altamente recomendado dedicar 4GB o más.

Notas

  • No es necesario descargar SKLauncher, si se tiene una cuenta Premium de Minecraft pueden descargar el Launcher Oficial

  • Si se registran y crean un usuario en la página de SKLauncher, podrán añadir un skin a su personaje que otras personas que utilicen el mismo Launcher podrán ver. Para descargar un skin, pueden utilzar una página como NameMC
